# Activity Tracking

iPhone uyumlu, PWA olarak ana ekrana eklenebilen bir üretkenlik takip uygulaması.

## Özellikler

- Varsayılan kategoriler: READING, ENGLISH, RUSSIAN, SKILLS, MEDITATION, SPORT, FINANCE.
- Haftalık / aylık / yıllık hedef belirleme.
- Pazartesi başlangıçlı hafta görünümü ve ileri/geri gezinme.
- Aylık görünümde ay isimleri ile ileri/geri gezinme.
- Birim bazlı gerçekleşme girişi (saat/dakika yok).
- Geçmiş tarihli kayıt ekleme ve silme.
- Kategori ekleme/silme, her kategori için görsel (emoji) desteği.
- Yeşil batarya doluluğu görünümü ile ilerleme takibi.
- Tüm kategorileri birleştiren üst özet gösterimi.

## Çalıştırma

```bash
python3 -m http.server 4173
```

Sonra tarayıcıda açın:

- Yerel: `http://localhost:4173/`
- GitHub Pages: `https://fkey7.github.io/Activity-Tracking/`
